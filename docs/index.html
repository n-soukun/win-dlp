FEATURES

主な特徴

WinDLPは、シンプルで直感的な操作感と、強力なダウンロード機能を両立しています。

簡単に使える

パソコンの初心者からプロフェッショナルまで、起動した瞬間から直感的に操作できるUI設計。難しい設定抜きで、すぐに動画ダウンロードを開始できます。

ダウンロードキュー機能

たくさんの動画をまとめて保存したい場合も、URLを入力してリストに追加していくだけ。あとはアプリがバックグラウンドで安全に自動処理します。

履歴機能

過去に取得した動画履歴を一覧管理。保存先フォルダへワンクリックでアクセスでき、ダウンロードしたファイルを見失う心配はありません。

Windows標準のデザイン

Windows 11のFluent Designに調和した美しさ。お使いのWindowsテーマやライト／ダークの個人設定にあわせて自動で最適に変化します。

HOW TO USE

使い方

わずか3ステップで、お気に入りの動画を保存できます。

01

WinDLPをダウンロード

本ページから最新のインストーラー（.msi）をダウンロードして実行し、インストールを完了させてアプリを起動します。

02

動画のURLをコピー

ブラウザなどで保存したい動画のページを開き、アドレスバーなどから動画のURLをコピーします。

03

貼り付けてダウンロード

コピーしたURLをWinDLPの入力欄に貼り付け、「動画を保存」ボタンをクリックするだけで保存が始まります。

システム要件

  • OS: Windows 10 (1809以降) / Windows 11 (64-bit)
  • Runtime: .NET Desktop Runtime 10.0以降
  • ネット接続: インターネット接続環境 (必須)

yt-dlpについて

WinDLPは、バックエンドとして yt-dlp を使用しています。アプリの初回起動時に最新のバイナリが自動的にダウンロードされるため、ユーザー側で個別の導入や設定作業は一切必要ありません。
