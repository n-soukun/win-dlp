# WinDLP

yt-dlpとffmpegを使って動画をダウンロード・変換するWindowsネイティブアプリです。

## 概要

WinDLPは、コマンドラインツールの `yt-dlp` および `ffmpeg` をグラフィカルユーザーインターフェース（GUI）から簡単に利用し、動画のダウンロードやフォーマット変換を行うことができるWindowsネイティブアプリケーションです。

## 使用方法

### ダウンロード

以下のリンクから最新のインストーラーをダウンロードできます。

*   [最新版インストーラーのダウンロード (windlp.msi)](https://github.com/n-soukun/win-dlp/releases/latest/download/windlp.msi)

### 機能紹介

*   **ダウンロード機能**: URLを貼り付けて「動画を保存」ボタンを押すことで、動画のダウンロードを開始します。
*   **履歴機能**: 過去にダウンロードした動画の履歴を表示・確認することができます。
